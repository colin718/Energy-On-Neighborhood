# Energy-On-Neighborhood
산업통상자원부 공공데이터 공모전 아이디어 부문 1차 합격작. 서울시 전력 데이터 기반 폭염 쉼터 안내 서비스

### 프로젝트 개요
**"폭염 취약계층을 위한 에너지 복지 & 쉼터 안내 플랫폼"**

**산업통상자원부 주관 공공데이터 활용 아이디어 공모전** 출품작으로, **1차 서류 합격 및 실무 임원진 대상 아이디어 부문 발표**를 진행했던 프로젝트의 웹 프로토타입입니다.
기후 위기로 인한 폭염 상황에서 서울시 전력 사용량 데이터와 기온 데이터를 분석하여, 에너지 빈곤층에게 **가까운 무더위 쉼터**와 **맞춤형 복지 혜택**을 안내하는 서비스를 기획 및 구현했습니다.

### 주요 성과 (Achievements)
* **공모전 성과:** 산업통상자원부 공공데이터 공모전 **1차 합격** (상위권 선정)
* **비즈니스 역량:** 공공데이터를 활용한 사회적 문제 해결 비즈니스 모델(BM) 수립 및 **임원진 대상 프레젠테이션 수행**

### 기술 스택 (Tech Stack)
* **Frontend:** `HTML5`, `CSS3` (Flashlight Effect, 3D Animations), `JavaScript (ES6+)`
* **Map Engine:** `Leaflet.js` (OpenStreetMap 기반 위치 정보 시각화)
* **UI/UX:**
    * **Interactive Flashlight:** 마우스 움직임에 반응하는 손전등 및 스포트라이트 효과 구현
    * **AI Chatbot UI:** 지원 사업 추천을 위한 대화형 인터페이스 디자인
    * **Responsive Design:** 반응형 웹 구조 설계

### 주요 기능 (Key Features)
1.  **동적 인터랙션 (Dynamic Interaction):** `CSS Keyframes`와 JS 이벤트를 활용한 고급 애니메이션 효과 (메인 화면 손전등 커서 등)
2.  **지도 데이터 시각화:** 서울시 25개 자치구청 좌표 데이터 매핑 및 마커(Marker) 바인딩
3.  **서비스 시나리오 구현:** 챗봇, 커뮤니티, 공지사항 등 실제 서비스 흐름을 고려한 UI 프로토타이핑

### 실행 방법 (How to Run)
`index.html` 파일을 다운로드하여 브라우저에서 실행해주세요.
